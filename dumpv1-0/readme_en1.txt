Installation:

Copy all files in a directory of your server.
Create a directory e.g. "backup" or let the script work :-) 
Open the file dump_cfg.inc in your favourite editor and change
the values to fit your environment.
de = deutsch, en = english
name backupfiles with date suffix: 0=no / 1=yes
(New Day = new file) 

Start the file "dump.php" with your webbrowser...
thats it.

-------------------------------------------------------------------------------------
FAQ:

> No connection with mysql database
- values at file dump_cfg.inc not ok.
